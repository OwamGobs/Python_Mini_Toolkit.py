#  Student Productivity App 

## ✨ Project Description

The **Student Productivity App** is a simple and interactive Python command-line application designed to help students stay productive, organised, and motivated every day.

This app combines:

*  A Study Planner
*  A Daily To-Do List
*  Mood-Based Motivation

The motivation feature is personalised based on the user’s mood, making the app feel more supportive and encouraging to the students.

---

#  Features

###  Study Planner

* Allows users to enter their school subjects
* Randomly generates a study schedule
* Suggests study hours for each subject

### To-Do List

* Displays random daily chores/tasks
* Helps students stay productive at home

### Mood-Based Motivation

* Asks the user how they are feeling
* Gives motivational quotes based on mood
* Provides small daily challenges
* Includes bonus productivity tips depending on the time and day
